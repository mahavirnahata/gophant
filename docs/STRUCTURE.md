# Repository Structure

- `auth/` Auth and gate.
- `cache/` Cache stores.
- `cli/` CLI helpers and runners.
- `cmd/gophant` CLI binary.
- `config/` Configuration loader.
- `container/` Dependency injection container.
- `db/` Query builder, schema, migrations, and ORM helpers.
- `http/` Router, context, and middleware types.
- `middleware/` Built‑in middleware.
- `queue/` Queue drivers and workers.
- `scheduler/` Scheduler.
- `security/` CSRF + password hashing.
- `session/` Session management.
- `testkit/` Testing helpers.
- `validation/` Request validation.
- `view/` Template rendering.
